# Apify Actors Monorepo

A comprehensive collection of Apify Actors for web scraping, data processing, and automation tasks. This monorepo contains multiple specialized actors built with TypeScript and follows Apify best practices.

## About Apify Actors

Apify Actors are serverless programs that run in isolated Docker containers. They're designed to:

- Accept well-defined JSON input
- Perform specific tasks (scraping, automation, data processing)
- Produce structured JSON output
- Scale automatically from seconds to hours
- Maintain state and support restarts
